Classy Laundry for Classy People

May 5, 2015

In order to demonstrate the broad utility of classes, I chose to make a class out of the first thing that popped into my head: A Laundry Machine! Speculation as to why I have nothing better to do midday on a Tuesday than laundry aside, laundry machines actually make pretty good object examples. Let's consider their properties:

  • Three-dimensional size,
  • They run on electricity (they have to be plugged in),
  • They can hold clothes,
  • They have a maximum capacity,
  • They require detergent (which has to be loaded), and
  • They wash clothes.

So how do we translate these attributes into a class? The first can easily be approximated with length, width, and height attributes. Whether the machine is plugged in makes a great boolean variable, as does the loading of detergent. Clothes is a little more complicated because there are many types of clothes, and we can't know in advance what type they will be or how many will be loaded into the machine. We only know the machine's capacity, which you probably guessed is easiest stored as an integer.

So here we have a perfect example of the benefits of classes. We don't need to worry about what type of clothes may be stuffed into our washing machine because those clothes themselves can be class objects, which will report to us their own attributes. By including size (an integer) and clean (a boolean) as attributes of the clothing class(es) and passing them to the washing machine, the machine can determine when it is full. The wash cycle can then be a function that sets all the clean variables to true. Below is my implementation of the washing machine class:

class Shirt
   attr_reader :size
  attr_accessor :clean

  def initialize
    @size = 5
    @clean = false
  end
end

class Pants
  attr_reader :size
  attr_accessor :clean
  def initialize
    @size = 4
    @clean = false
  end
end

class Socks
  attr_reader :size
  attr_accessor :clean
  def initialize
    @size = 1
    @clean = false
  end
end

class Washing_Machine
  def initialize(length, width, height, capacity)
    @length = length
    @width = width
    @height = height
    @capacity = capacity
    @plugged_in = true
    @detergent_loaded = false
    @contents = []
  end

  attr_reader :length, :width, :height
  attr_accessor :plugged_in, :detergent_loaded

  def state
    @contents.inject(0){|sum, n| sum + n.size}
  end

  def remaining_capacity
    @capacity - self.state
  end

#input clothes must be an array of clothing objects
  def load(clothes)
    @contents += clothes
  end
  def cycle
    if @contents.empty?
      p "There are no clothes in the machine to wash."
    else
      @contents.each {|i| i.clean = true}
    end
  end

  def empty
    output = @contents.dup
    @contents.clear
    output
  end
end
